# EBT Instructor Features

Comprehensive feature overview for EBT Instructor.

## Session Management

### Pre-Session Planning

- Access trainee history and records
- Select appropriate scenarios
- Review training objectives
- Prepare assessment criteria

### During Session

- Real-time performance monitoring
- Evidence capture tools
- Immediate feedback capabilities
- Assessment scoring system

### Post-Session

- Detailed session reports
- Evidence documentation
- Performance analysis
- Record submission

## Assessment Tools

### Performance Evaluation

- Structured assessment forms
- Rubric-based scoring
- Evidence-based evaluation
- Competency tracking

### Feedback Mechanisms

- Real-time feedback
- Post-session debriefing tools
- Improvement recommendations
- Action plan development

## Integration Features

- Direct connection to Aero EBT
- Scenario import from Scenario Builder
- Data export capabilities
- Report generation

## Collaboration

- Share sessions with other instructors
- Peer review capabilities
- Team training management
- Communication tools
