# EBT Scenario Builder User Guide

Complete guide to using EBT Scenario Builder effectively.

## Interface Overview

### Scenario Editor

The main editor provides:
- Component palette
- Canvas area
- Properties panel
- Preview pane

### Navigation

- **My Scenarios**: Access your scenarios
- **Templates**: Browse template library
- **Shared**: View shared scenarios
- **Library**: Public scenario library

## Building Scenarios

### Scenario Structure

1. **Scenario Information**: Basic details and metadata
2. **Events**: Training events and triggers
3. **Conditions**: Environmental and situational settings
4. **Objectives**: Learning goals and competencies
5. **Assessment**: Evaluation criteria

### Adding Events

1. Drag event from palette
2. Configure event properties
3. Set triggers and conditions
4. Define event outcomes

### Setting Objectives

1. Add learning objectives
2. Map to competencies
3. Define success criteria
4. Link to assessment

## Best Practices

1. **Clear Objectives**: Define clear, measurable objectives
2. **Realistic Scenarios**: Keep scenarios realistic and relevant
3. **Progressive Difficulty**: Build appropriate challenge levels
4. **Evidence Focus**: Design for evidence collection
5. **Review and Test**: Always test scenarios before deployment

## Tips & Tricks

- Use templates as starting points
- Reuse successful scenario components
- Collaborate with other instructors
- Keep scenarios updated
- Document scenario rationale

## Troubleshooting

### Common Issues

- **Validation Errors**: Review error messages and fix issues
- **Save Problems**: Check network connection
- **Import Issues**: Verify file format and version
