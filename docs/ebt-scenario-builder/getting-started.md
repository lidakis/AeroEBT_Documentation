# Getting Started with EBT Scenario Builder

Learn how to create your first training scenario.

## Prerequisites

- Access to EBT Scenario Builder
- Understanding of EBT principles
- Clear training objectives

## Creating Your First Scenario

### Step 1: Start a New Scenario

1. Log in to EBT Scenario Builder
2. Click **New Scenario**
3. Choose to start from:
   - Blank template
   - Existing template
   - Copy of existing scenario

### Step 2: Define Basic Information

- **Scenario Name**: Give your scenario a descriptive name
- **Description**: Provide context and objectives
- **Training Level**: Specify appropriate training level
- **Duration**: Set expected scenario duration

### Step 3: Build Scenario Components

- **Events**: Add training events and triggers
- **Conditions**: Define environmental and situational conditions
- **Objectives**: Set learning objectives and competencies
- **Assessment Criteria**: Define evaluation standards

### Step 4: Test and Validate

- Review scenario flow
- Test event triggers
- Validate against training objectives
- Check for consistency

### Step 5: Save and Deploy

- Save your scenario
- Publish to scenario library
- Make available to instructors
- Integrate with Aero EBT

## Scenario Templates

Browse the template library for:
- Standard EBT scenarios
- Common training situations
- Regulatory compliance scenarios
- Customizable base scenarios
