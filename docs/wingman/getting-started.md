# Getting Started with Wingman

Set up Wingman for your flight operations.

## Prerequisites

- Compatible flight system or device
- Wingman software installation
- User account and credentials
- Initial configuration completed

## Initial Setup

### Step 1: Installation

1. Install Wingman software on your device
2. Launch the application
3. Complete initial setup wizard
4. Verify system compatibility

### Step 2: Configuration

1. **Connect to Flight System**: Establish connection to avionics
2. **User Profile**: Set up pilot profile and preferences
3. **Alert Settings**: Configure safety alert thresholds
4. **Integration**: Connect to Aero EBT if available

### Step 3: First Flight

1. Pre-flight: Verify Wingman status
2. During flight: Monitor Wingman displays
3. Post-flight: Review flight data

## Basic Operations

### Monitoring

- View real-time flight parameters
- Monitor safety indicators
- Check alert status
- Review recommendations

### Alerts

Wingman provides alerts for:
- Safety-critical situations
- Performance deviations
- System anomalies
- Operational recommendations

### Data Collection

- Automatic flight data recording
- Performance metrics capture
- Event logging
- Data synchronization
