# Getting Started with EBT Instructor

This guide will help instructors set up and start using EBT Instructor.

## First-Time Setup

### Prerequisites

- Instructor credentials provided by your administrator
- Access to Aero EBT platform
- Instructor certification and qualifications

### Initial Configuration

1. **Login**: Access EBT Instructor using your credentials
2. **Profile Setup**: Complete your instructor profile
3. **Certifications**: Verify and update your certifications
4. **Preferences**: Configure session and assessment preferences

### Step 2: Conduct the Session

- Monitor trainee performance in real-time
- Record observations and evidence
- Provide immediate feedback when appropriate
- Use assessment tools for evaluations

### Step 3: Complete and Review

- Finalize assessments
- Review session data
- Submit to training records
- Schedule follow-up if needed

## Quick Tips

- Familiarize yourself with the interface before your first session
- Use the built-in checklists for consistent evaluations
- Take advantage of scenario templates
- Review trainee history before sessions
