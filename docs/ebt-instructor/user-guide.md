# EBT Instructor User Guide

Detailed guide for instructors using EBT Instructor.

## Interface Overview

### Main Dashboard

- Active sessions overview
- Upcoming scheduled sessions
- Recent assessments
- Quick access tools

### Session Interface

The session interface includes:
- Trainee information panel
- Real-time monitoring tools
- Assessment forms
- Evidence capture tools
- Communication panel

## Conducting a Session

### Before the Session

1. Review trainee history
2. Prepare scenario and objectives
3. Check system and equipment
4. Review assessment criteria

### During the Session

1. Monitor performance continuously
2. Record observations and evidence
3. Use assessment tools appropriately
4. Provide feedback as needed

### After the Session

1. Complete assessments
2. Document evidence
3. Generate session report
4. Submit to training records

## Best Practices

1. **Preparation**: Always review trainee history before sessions
2. **Evidence**: Document evidence as it occurs
3. **Feedback**: Provide balanced, constructive feedback
4. **Consistency**: Use standard assessment criteria
5. **Follow-up**: Schedule follow-up sessions when needed

## Troubleshooting

### Common Issues

- **Connection Problems**: Check network connection and system status
- **Data Sync**: Ensure proper sync with Aero EBT
- **Performance Issues**: Clear cache or restart application
